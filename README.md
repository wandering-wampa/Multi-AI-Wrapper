# Multi-AI Wrapper

Multi-AI Wrapper is a lightweight Electron-based desktop app that puts multiple AI assistants into a single window.

Supported services:
- ChatGPT
- Claude
- Copilot
- Gemini
- Perplexity

Features:
- Simple tabbed interface
- Lazy-loaded views for faster switching after first load
- Windows build ready; macOS build generated via GitHub Actions

Built collaboratively with the assistance of AI tools.
